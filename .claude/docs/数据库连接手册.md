## 架构概述

nl2sql-langgraph 采用**双数据库架构**：

| 数据库 | 用途 | 驱动 | 核心表 |
|--------|------|------|--------|
| PostgreSQL | 问数业务数据 | `psycopg` | `enterprise_kb`, `metrics_catalog`, `lake_table_metadata`, `fact_orders` |
| MySQL | 平台数据 | `PyMySQL` | `chat_session`, `query_history` |

```
┌─────────────────┐     ┌─────────────────┐
│   PostgreSQL    │     │     MySQL       │
│  (问数业务库)    │     │   (平台数据库)   │
├─────────────────┤     ├─────────────────┤
│ enterprise_kb   │     │ chat_session    │
│ metrics_catalog │     │ query_history   │
│ lake_table_     │     │ user_preferences│
│   metadata      │     │                 │
│ fact_orders     │     │                 │
└─────────────────┘     └─────────────────┘
        ↑                       ↑
        │                       │
   app/tools.py           app/mysql_tools.py
```

## PostgreSQL 连接

### 连接配置

```env
POSTGRES_DSN=postgresql://postgres:password@localhost:5432/nl2sql
```

### 连接方式

```python
# app/tools.py
import psycopg

def get_postgres_dsn() -> str:
    return os.getenv("POSTGRES_DSN")

def execute_sql(sql: str) -> dict:
    dsn = get_postgres_dsn()
    with psycopg.connect(dsn) as conn:
        with conn.cursor() as cur:
            cur.execute(sql)
            rows = cur.fetchall()
            columns = [desc[0] for desc in (cur.description or [])]
    return {"columns": columns, "rows": [list(r) for r in rows]}
```

### 检索函数

| 函数 | 用途 | 返回 |
|------|------|------|
| `fetch_knowledge_hits(keywords)` | 企业知识库检索 | 匹配的知识条目 |
| `fetch_metrics_hits(keywords)` | 指标目录检索 | 匹配的指标定义 |
| `fetch_metadata_hits(topics)` | 元数据检索 | 匹配的表结构 |

### 查询示例

```python
# 使用 dict_row 获取字典结果
from psycopg.rows import dict_row

with psycopg.connect(dsn) as conn:
    with conn.cursor(row_factory=dict_row) as cur:
        cur.execute("SELECT * FROM enterprise_kb LIMIT 5")
        rows = cur.fetchall()
        # rows: [{"id": 1, "topic": "订单", ...}, ...]
```

```python
# app/mysql_tools.py
import pymysql
from pymysql.cursors import DictCursor

def get_mysql_connection():
    return pymysql.connect(
        host=os.getenv("MYSQL_HOST", "localhost"),
        port=int(os.getenv("MYSQL_PORT", "3306")),
        user=os.getenv("MYSQL_USER", "root"),
        password=os.getenv("MYSQL_PASSWORD", ""),
```

## 安全机制

### SQL 注入防护

PostgreSQL 使用**参数化查询**：

```python
# ✅ 安全：参数化查询
sql = """
    SELECT * FROM enterprise_kb
    WHERE lower(keyword_synonyms) LIKE ANY (%s)
"""
cur.execute(sql, (patterns,))

# ❌ 危险：字符串拼接
sql = f"SELECT * FROM enterprise_kb WHERE keyword = '{keyword}'"
```

### SQL 语句校验

```python
def safe_sql_validate(sql: str, sql_max_rows: int) -> tuple[str, str | None]:
    """安全校验：只允许 SELECT，禁止多语句"""
    s = sql.strip().rstrip(";")

    # 禁止多语句
    if ";" in s:
        raise ValueError("SQL 不允许多语句")

    # 只允许 SELECT
    if not s.lower().startswith("select"):
        raise ValueError("仅允许 SELECT 语句")

    # 自动添加 LIMIT
    if not re.search(r"\blimit\b", s.lower()):
        s = f"{s} LIMIT {sql_max_rows}"
        return s, "已自动补充 LIMIT"

    return s, None
```

## 连接池优化

### 当前实现（短连接）

每次操作创建新连接，操作完成后关闭：

```python
def execute_sql(sql: str) -> dict:
    with psycopg.connect(dsn) as conn:  # 每次新建连接
        with conn.cursor() as cur:
            cur.execute(sql)
            return {"columns": ..., "rows": ...}
    # 自动关闭连接
```

### 生产环境建议（连接池）

```python
# 使用 psycopg_pool 连接池
from psycopg_pool import ConnectionPool

pool = ConnectionPool(dsn, min_size=2, max_size=10)

def execute_sql(sql: str) -> dict:
    with pool.connection() as conn:  # 从池中获取
        with conn.cursor() as cur:
            cur.execute(sql)
            return {"columns": ..., "rows": ...}
    # 归还到池中
```

```powershell

psql -U postgres -c "CREATE DATABASE nl2sql;"

# 执行建表脚本
psql $env:POSTGRES_DSN -f .\db\schema.sql

# 导入种子数据
psql $env:POSTGRES_DSN -f .\db\seed.sql
```

```powershell
# 创建数据库
mysql -u root -p -e "CREATE DATABASE nl2sql_platform;"

# 执行建表脚本
mysql -u root -p nl2sql_platform < .\db\mysql_schema.sql
```

## 故障排查

### PostgreSQL 连接失败

**错误**：`could not connect to server: Connection refused`

**检查**：
1. PostgreSQL 服务是否启动
2. 端口 5432 是否开放
3. `pg_hba.conf` 是否允许连接

**解决**：
```powershell
# Windows
net start postgresql-x64-16

# Linux
sudo systemctl start postgresql
```

### MySQL 连接失败

**错误**：`Can't connect to MySQL server on 'localhost'`

**检查**：
1. MySQL 服务是否启动
2. 用户名/密码是否正确
3. 数据库是否已创建

**解决**：
```powershell
# Windows
net start mysql

# Linux
sudo systemctl start mysql
```

### 编码问题

**错误**：中文显示乱码

**解决**：
```python
# PostgreSQL（UTF-8 默认）
# 无需额外配置

# MySQL
conn = pymysql.connect(
    ...,
    charset="utf8mb4",  # 指定编码
)
```

## 数据库版本要求

| 数据库 | 最低版本 | 推荐版本 |
|--------|---------|---------|
| PostgreSQL | 12.0 | 15.0+ |
| MySQL | 5.7 | 8.0+ |

### 版本特性依赖

- **PostgreSQL**：`ILIKE ANY (ARRAY[...])` 语法（12.0+）
- **MySQL**：`utf8mb4` 编码（5.5.3+，8.0 默认）
